# TweetReview

## EN

```markdown
# Tweet Review

Review the draft tweet and provide feedback based on the following criteria:

1. **Clarity**: Ensure the tweet is easy to understand, and suggest improvements if necessary.
2. **Tone**: Check if the tweet aligns with the intended tone, and propose adjustments if needed.
3. **Politeness**: Ensure the tweet does not contain offensive or inappropriate language, and provide corrections if necessary.
4. **Grammar and Spelling**: Correct any grammatical or spelling mistakes.
5. **Potential for Misunderstanding**: Ensure the tweet does not contain any wording that could lead to misunderstandings, and suggest corrections if needed.
```

## JP

```markdown
# ツイートレビュー

ツイートのドラフトを受け取り、以下の点に基づいてフィードバックを行います。

1. **明確さ**：ツイートが分かりやすいかを確認し、必要なら改善案を提案してください。
2. **トーン**：意図したトーンに合っているかを確認し、適切な改善案を提案してください。
3. **礼儀正しさ**：攻撃的な表現や不適切な言葉がないか確認し、修正案を提案してください。
4. **文法とスペル**：文法やスペルの誤りを修正してください。
5. **誤解の可能性**：ツイートが誤解を生む表現を含んでいないか確認し、必要なら修正を提案してください。
```
